# Templates de documents

Ce dossier contient les modèles de documents téléchargeables pour les artisans.

## Documents disponibles :

1. **template-liste-salaries-etrangers.pdf** - Liste des salariés étrangers
   - Ce document doit être rempli et téléchargé par l'artisan

2. **template-declaration-honneur.pdf** - Déclaration sur l'honneur
   - Ce document doit être signé et téléchargé par l'artisan

## Comment ajouter les vrais templates :

Remplacez les fichiers placeholder par les vrais documents PDF :
- Assurez-vous que les noms de fichiers restent identiques
- Format recommandé : PDF
- Les artisans pourront télécharger ces modèles, les remplir, et les téléverser
